说明

class="sred"：

选中星星的颜色变成红色

class="ax"：

星星总(x)个数，目前有3|5|8|10，四种长度。其中五星有jquery.rating.js插件的支持

class="sxxx"：

选中星星总(x)个数，目前为0~9.5，代表选中(select)0~选中(select)9.5颗星；其中class="s0d5"代表选中半星

class="sall"：

选中所有星星，无论星星总个数是多少都全部选中
